# svkdb

The p3ng0s fork of svkdb ^^
This is mainly built for steam deck usage you can press the `Volume-Down` button
and get this menu popup from DWM on any machine.
This fork comes with a application launcher in the 3rd screenshot. This app launcher
is just a combination of all of the dwm shortcuts set to a single button press.
